Job Emailer

Command-line interface to automate sending job application emails with attachments.

Linux Installation (Debian/Ubuntu)

Step 1: Add the Repository Key
curl -fsSL [URL]/KEY.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/job-emailer.gpg
Step 2: Add the Repository
echo "deb [arch=all] [URL] /" | sudo tee /etc/apt/sources.list.d/job-emailer.list
Step 3: Install
sudo apt update && sudo apt install job-emailer
Note: The commands above verify that you are connecting to the correct repository hosted at this location.
